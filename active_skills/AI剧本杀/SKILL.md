# AI剧本杀

## 概述

回合制异世界人生模拟游戏。玩家穿越到随机生成的异世界，从出生开始，每回合做出选择，经历一生，见证世界变迁。

**核心玩法**：
- 🌍 随机生成世界观（魔法体系、种族、国家、历史）
- 👤 随机出身（种族、家庭、天赋、命运）
- ⏳ 回合制推进（每回合=1年或特定阶段）
- 🎲 事件选择（随机事件+命运事件+玩家选择）
- 📖 故事生成（自动记录人生故事）
- 🖼️ 图片生成（关键场景、角色卡）
- 📚 终局成书（人生结束时生成完整短篇小说）

## 输出格式规范

### 核心原则：游戏输出 + 小说同步

**游戏以游戏形式推进，同时生成小说文件。**

1. **游戏输出** — 使用判定框、选择框等游戏元素，保持交互性
2. **小说同步** — 每次推进剧情后，同步更新小说文件，用小说语言重写
3. **小说文件路径** — `stories/[角色名]_[世界名]_[日期]/[角色名]的故事.md`
4. **小说风格** — 第三人称限制视角，沉浸式叙事，判定结果融入故事

### 游戏输出格式

保持原有的游戏风格：

---

**第 {回合数} 回合 · {年龄}岁 · {季节} · {地点}**

---

> **📜 {事件标题}**
>
> {事件描述...}

**【判定】**

| 判定 | 结果 |
|------|------|
| 属性 | {属性名} |
| 需求 | ≥ {数值} |
| 你的值 | {当前值} ✓/✗ |

**【选择】**

| 选项 | 行动 |
|------|------|
| [1] | {选项1} |
| [2] | {选项2} |

## 游戏流程

```
新游戏 → 世界观生成 → 生成3个角色选项 → 玩家选择角色 → 第一回合
    ↓
回合循环：
    显示状态 → 生成事件 → 玩家选择 → 执行结果 → 记录故事 → 下一回合
    ↓
人生结束 → 终局成书 → 可选：重新开始
```

## 世界观生成规则

### 核心原则：自由创造

**AI应当自由创造世界观，而非局限于预设类型。**

每次新游戏，AI根据随机种子或灵感，创造一个独特的世界观：

1. **世界类型** — 可以是任何想象得到的类型（奇幻、科幻、现实架空、混合...）
2. **力量体系** — 可以是魔法、修仙、超能力、科技、灵异、或者完全原创的体系
3. **种族设定** — 可以是人类、外星人、妖怪、机器人、或者从未见过的种族
4. **社会结构** — 可以是帝国、联邦、部落、公司、宗门、或者全新的组织形式
5. **核心矛盾** — 世界面临的主要冲突或挑战

## 角色成长体系

### 核心原则

角色在游戏过程中会不断成长，通过**事件完成、选择结果、时间流逝**获得属性提升、技能、特质和物品。

### 属性成长规则

| 成长方式 | 说明 | 示例 |
|----------|------|------|
| **自然成长** | 根据人生阶段自动成长 | 少年期智力+2，青年期力量+1 |
| **事件奖励** | 完成事件后获得属性提升 | 战斗胜利→力量+1 |
| **判定成功** | 判定成功可能获得微量成长 | 敏捷判定成功→敏捷+0.5 |
| **特质影响** | 特质影响成长速度 | 「体弱」→力量成长-20% |
| **训练/学习** | 专门花时间训练 | 修行一个月→相关属性+1~3 |

**属性上限：** 每项属性上限为20，超过后以「突破」形式记录。

**属性评价对照：**

| 数值范围 | 评价 |
|----------|------|
| 1-5 | 极弱/迟钝/愚笨 |
| 6-8 | 较弱/普通/平庸 |
| 9-11 | 普通/正常/一般 |
| 12-14 | 良好/优秀/出色 |
| 15-17 | 优秀/天才/卓越 |
| 18-20 | 极限/传说/神话 |

### 技能获取规则

| 获取方式 | 说明 |
|----------|------|
| **觉醒** | 出生或特定事件自动获得 |
| **学习** | 花时间学习（需要条件和资源） |
| **事件奖励** | 完成特定事件获得 |
| **传承** | 由他人传授 |
| **突破** | 属性达到阈值自动解锁 |

**技能等级：**
- 入门 → 熟练 → 精通 → 大师 → 传说

**技能示例：**

| 世界类型 | 技能示例 |
|----------|----------|
| 奇幻 | 剑术、魔法、炼金、骑术 |
| 修仙 | 御剑、符箓、丹道、阵法 |
| 科幻 | 黑客、机甲驾驶、基因改造 |
| 都市 | 格斗、驾驶、黑客、社交 |
| 武侠 | 轻功、内功、暗器、医术 |

### 特质获取规则

| 类型 | 获取方式 |
|------|----------|
| **初始特质** | 创建角色时随机/选择获得 |
| **事件特质** | 经历重大事件后获得 |
| **成长特质** | 属性达到阈值解锁 |
| **创伤特质** | 经历严重创伤/失败获得 |
| **蜕变特质** | 完成特殊任务/突破获得 |

**特质示例：**

| 类型 | 特质 | 效果 |
|------|------|------|
| ✓ 正面 | **铁意志** | san值上限+20，抵抗精神攻击+30% |
| ✓ 正面 | **天赋异禀** | 选择一项属性，成长速度+50% |
| ✗ 负面 | **创伤后应激** | 特定场景san值-10，闪避-2 |
| ✗ 负面 | **诅咒之体** | 恢复速度-50%，需要特殊方法治疗 |
| ◆ 中性 | **命运之子** | 幸运事件概率+30%，但也会吸引危险 |
| ◆ 中性 | **双重人格** | 可切换人格，不同人格属性/技能不同 |

### 物品/道具系统

| 类型 | 说明 | 示例 |
|------|------|------|
| **装备** | 提供属性加成 | 武器、护甲、饰品 |

**物品获取方式：**

## 世界发展体系

### 核心原则

世界不是静止的，会随着时间推移和玩家行动而变化。重大事件会影响世界格局，玩家的选择可能改变世界走向。

### 时间推进规则

| 回合类型 | 时间跨度 | 说明 |
|----------|----------|------|
| 婴儿期 | 1回合 = 1岁 | 被动成长为主 |
| 童年期 | 1回合 = 1年 | 学习探索为主 |
| 少年/青年期 | 1回合 = 数月~1年 | 根据事件调整 |
| 冒险期 | 1回合 = 数天~数周 | 紧张刺激 |

### 世界事件系统

| 事件类型 | 触发方式 | 影响 |
|----------|----------|------|
| **周期事件** | 固定时间触发 | 季节变化、节日、自然灾害 |
| **随机事件** | 概率触发 | 瘟疫、发现、意外 |
| **因果事件** | 玩家选择触发 | 战争、革命、科技进步 |
| **命运事件** | 剧情节点 | 重大转折、Boss战、世界危机 |

### 世界状态追踪

**每回合更新：**

| 项目 | 说明 |
|------|------|
| 年份 | 当前世界年份 |
| 季节 | 春夏秋冬（影响事件） |
| 世界状态 | 和平/动荡/战争/灾难 |
| 势力变化 | 各势力兴衰 |
| 科技/魔法水平 | 文明发展程度 |
| 公共事件 | 所有人都会受影响的大事 |

**世界状态输出格式：**

**【世界动态】**

| 项目 | 变化 |
|------|------|
| 年份 | 1248年（+1） |
| 季节 | 春 → 夏 |
| 大事 | 北方出现不明梦魇潮 |
| 影响 | 商路受阻，物价上涨 |

### 玩家对世界的影响

**影响力系统：**

| 影响力等级 | 效果 |
|------------|------|
| 无名小卒 | 无法影响世界事件 |
| 小有名气 | 可以参与一些事件 |
| 声名远扬 | 选择可以改变局部局势 |
| 举足轻重 | 个人选择影响世界走向 |
| 传奇人物 | 改写历史，创造传说 |

**影响方式：**
- 完成重要任务
- 击败强敌
- 做出重大选择
- 积累声望/财富/权力

### 势力系统

**势力属性：**

| 属性 | 说明 |
|------|------|
| 实力 | 军事/经济/魔法力量 |
| 领土 | 控制范围 |
| 关系 | 与其他势力的外交 |
| 玩家关系 | 玩家在势力中的地位 |

**势力动态：**
- 势力之间会发生战争、结盟、吞并
- 玩家可以选择加入或对抗某势力
- 势力变化会影响玩家可接的任务

## 事件奖励设计指南

### 奖励设计原则

1. **合理平衡** — 奖励与难度匹配
2. **多样选择** — 不同选择给予不同奖励
3. **长远影响** — 部分奖励延迟生效
4. **意外惊喜** — 偶尔给予超预期奖励

### 奖励池设计

**根据事件类型设计奖励：**

| 事件类型 | 常见奖励 |
|----------|----------|
| 战斗事件 | 属性+1、技能碎片、装备、金钱 |
| 探索事件 | 线索、物品、属性+0.5、隐藏地点解锁 |
| 社交事件 | 关系提升、信息、称号、机会 |
| 学习事件 | 技能、属性+1、知识、特质 |
| 危机事件 | san值变化、特质、属性变化、关键物品 |

### 失败也有收获

失败不应一无所获：

| 失败类型 | 收获 |
|----------|------|
| 勉强失败 | 经验值、微量属性、教训（特质） |
| 惨痛失败 | 创伤特质、关键线索、人际关系变化 |
| 致命失败 | 重生机会、传承、剧情转折 |

### 奖励输出时机

1. **即时奖励** — 事件结束后立即显示
2. **延迟奖励** — 数回合后触发效果
3. **条件奖励** — 满足条件后激活
4. **隐藏奖励** — 不显示，后续剧情揭示

## 事件系统

### 事件类型

1. **随机事件** — 根据概率触发
2. **命运事件** — 预设的人生节点
3. **世界事件** — 全球性大事
4. **关系事件** — 人际关系相关
5. **选择事件** — 玩家主动选择触发

### 事件权重

事件触发考虑：
- 玩家属性（智力高→学习事件多）
- 人生阶段（少年期→教育事件）
- 世界状态（战争时期→征兵事件）
- 关系网络（贵族朋友→社交事件）
- 前置事件（之前的选择影响后续）

## 终局成书

### 触发条件

当角色死亡或人生结束时，自动进入「终局成书」流程，将整个游戏过程生成一部完整的短篇小说。

### 成书流程

```
人生结束 → 统计数据 → 生成章节大纲 → 撰写小说 → 生成封面 → 导出文件
```

### 小说结构

**标准章节结构：**

| 章节 | 内容 |
|------|------|
| 序章 | 世界观背景、出生 |
| 第1章 | 婴儿期（0-3岁） |
| 第2章 | 童年期（4-12岁） |
| 第3章 | 少年期（13-17岁） |
| 第4章 | 青年期（18-35岁） |
| 第5章 | 中年期（36-55岁） |
| 第6章 | 老年期（56岁+） |
| 尾声 | 死亡、遗产、后世评价 |
| 附录 | 属性记录、成就列表、关键选择 |

### 写作风格

根据世界观自动调整：

| 世界类型 | 风格 | 特点 |
|----------|------|------|
| 奇幻中世纪 | 古典优雅 | 诗意描写、史诗感 |
| 东方玄幻 | 空灵飘逸 | 古风语言、意境深远 |
| 蒸汽朋克 | 机械硬朗 | 工业感、齿轮意象 |
| 黑暗奇幻 | 压抑阴郁 | 克苏鲁风、悬疑感 |
| 异能都市 | 现代简洁 | 快节奏、都市感 |
| 废土末世 | 粗砺真实 | 生存感、荒凉 |

### 叙事视角

- **主视角：** 第三人称有限视角（聚焦主角）
- **重要NPC：** 可切换视角补充剧情
- **关键选择：** 第一人称内心独白

### 输出文件

**生成文件：**

```
stories/[角色名]_[世界名]_[日期]/
├── [角色名]的一生.md        # 完整小说（Markdown）
├── [角色名]的一生.txt       # 纯文本版本
├── 封面.png                 # 小说封面（角色终局图）
├── data/
│   ├── world.json           # 世界数据
│   ├── player.json          # 角色数据
│   └── history.json         # 完整事件历史
└── images/
    └── [所有生成的图片]
```

### 小说示例

**序章示例：**

> 艾尔德兰大陆，一个剑与魔法的世界。
>
> 三百年前，黑暗魔王降临，五大种族联手将其封印。
> 一百年前，魔法开始衰退，无人知晓缘由。
>
> 1247年，初春，河湾镇。
>
> 一个女婴降生于此。她有着黑色的卷发和银色的眼眸，体内流淌着人类与精灵的血液。
>
> 她的母亲给她取名——艾莉丝。
>
> 这个名字，将被写入历史。

**尾声示例：**

> 艾莉丝闭上眼睛的那一刻，想起了母亲的话：
>
> "你的天赋，是礼物，也可能是诅咒。"
>
> 她的一生，证明了这句话。她用智慧改变了命运，用勇气对抗了黑暗，用爱守护了家人。
>
> 她的故事，在艾尔德兰大陆流传了百年。
>
> 而命簿，翻到了最后一页。

| 命令 | 说明 |

| 传奇版 | 突出戏剧性，像传奇故事 |
| 编年史版 | 客观记录，像历史文献 |
