# AI剧本杀

## 概述

回合制异世界人生模拟游戏。玩家穿越到随机生成的异世界，从出生开始，每回合做出选择，经历一生，见证世界变迁。

**核心玩法**：
- 🌍 随机生成世界观（魔法体系、种族、国家、历史）
- 👤 随机出身（种族、家庭、天赋、命运）
- ⏳ 回合制推进（每回合=1年或特定阶段）
- 🎲 事件选择（随机事件+命运事件+玩家选择）
- 📖 故事记录（自动记录人生故事）
- 🖼️ 图片生成（关键场景、角色卡）
- 📚 终局成书（人生结束时可选生成小说）

**设计理念**：游戏以游戏形式推进，保持强交互性。小说是可选的终局产物，不是游戏过程。

## 游戏流程

```
新游戏 → 世界观生成 → 生成3个角色选项 → 玩家选择角色 → 第一回合
    ↓
回合循环：
    显示状态 → 生成事件 → 玩家选择 → 判定执行 → 显示结果/奖励 → 下一回合
    ↓
人生结束 → 可选：终局成书 → 可选：重新开始
```

## 初始角色选择

### 生成规则

新游戏开始时，**必须生成3个不同的角色选项**，让玩家选择：

1. 三个角色在世界观内**种族、出身、天赋各不相同**
2. 每个角色都要**完整描述性别、外貌、属性、特质**
3. 玩家选择后，进入该角色的人生

### 输出格式

## 角色成长体系

### 属性成长规则

| 成长方式 | 说明 | 示例 |
|----------|------|------|
| 自然成长 | 根据人生阶段自动成长 | 少年期智力+2，青年期力量+1 |
| 事件奖励 | 完成事件后获得属性提升 | 战斗胜利→力量+1 |
| 判定成功 | 判定成功可能获得微量成长 | 敏捷判定成功→敏捷+0.5 |
| 特质影响 | 特质影响成长速度 | 「体弱」→力量成长-20% |
| 训练/学习 | 专门花时间训练 | 修行一个月→相关属性+1~3 |

**属性上限：** 每项属性上限为20。

**属性评价对照：**

| 数值 | 评价 |
|------|------|
| 1-5 | 极弱/愚笨 |
| 6-8 | 较弱/平庸 |
| 9-11 | 普通/一般 |
| 12-14 | 良好/出色 |
| 15-17 | 优秀/天才 |
| 18-20 | 极限/传说 |

### 技能获取规则

| 获取方式 | 说明 |
|----------|------|
| 觉醒 | 出生或特定事件自动获得 |
| 学习 | 花时间学习（需要条件和资源） |
| 事件奖励 | 完成特定事件获得 |
| 传承 | 由他人传授 |
| 突破 | 属性达到阈值自动解锁 |

**技能等级：** 入门 → 熟练 → 精通 → 大师 → 传说

### 特质获取规则

| 类型 | 获取方式 |
|------|----------|
| 初始特质 | 创建角色时随机/选择获得 |
| 事件特质 | 经历重大事件后获得 |
| 成长特质 | 属性达到阈值解锁 |
| 创伤特质 | 经历严重创伤/失败获得 |

**特质示例：**

| 类型 | 特质 | 效果 |
|------|------|------|
| ✓ 正面 | 铁意志 | san值上限+20，抵抗精神攻击+30% |
| ✓ 正面 | 天赋异禀 | 选择一项属性，成长速度+50% |
| ✗ 负面 | 创伤后应激 | 特定场景san值-10，闪避-2 |
| ✗ 负面 | 诅咒之体 | 恢复速度-50% |
| ◆ 中性 | 命运之子 | 幸运事件+30%，但也会吸引危险 |

## 世界发展体系

### 时间推进规则

| 回合类型 | 时间跨度 | 说明 |
|----------|----------|------|
| 婴儿期 | 1回合 = 1岁 | 被动成长为主 |
| 童年期 | 1回合 = 1年 | 学习探索为主 |
| 少年/青年期 | 1回合 = 数月~1年 | 根据事件调整 |

### 世界状态追踪

### 玩家对世界的影响

| 影响力等级 | 效果 |
|------------|------|

| 小有名气 | 可以参与一些事件 |
| 声名远扬 | 选择可以改变局部局势 |
| 举足轻重 | 个人选择影响世界走向 |
| 传奇人物 | 改写历史，创造传说 |

## 事件系统

### 事件类型

| 类型 | 触发方式 | 影响 |
|------|----------|------|
| 随机事件 | 概率触发 | 瘟疫、发现、意外 |
| 命运事件 | 剧情节点 | 重大转折、Boss战 |
| 世界事件 | 固定时间触发 | 战争、革命、天灾 |
| 关系事件 | 人际触发 | 社交、冲突、合作 |

### 事件权重考虑

- 玩家属性（智力高→学习事件多）
- 人生阶段（少年期→教育事件）
- 世界状态（战争时期→征兵事件）
- 关系网络（贵族朋友→社交事件）
- 前置事件（之前的选择影响后续）

### 事件奖励设计

| 事件类型 | 常见奖励 |
|----------|----------|
| 战斗事件 | 属性+1、技能碎片、装备、金钱 |
| 探索事件 | 线索、物品、属性+0.5、隐藏地点解锁 |
| 社交事件 | 关系提升、信息、称号、机会 |
| 学习事件 | 技能、属性+1、知识、特质 |
| 危机事件 | san值变化、特质、属性变化、关键物品 |
